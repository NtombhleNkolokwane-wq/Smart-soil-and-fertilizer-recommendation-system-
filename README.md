
# Smart Soil & Fertilizer Recommendation System

This project is a web-based agricultural decision support system developed using Python, Flask, and MySQL. The system analyzes soil information and provides fertilizer recommendations to help farmers improve crop productivity and soil health.

## Features
- Soil data management
- Fertilizer recommendation generation
- Database-driven storage and retrieval
- User-friendly web interface
- Agricultural decision support

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS

## Purpose
The project aims to support sustainable agriculture by helping farmers make informed fertilizer decisions based on soil conditions.
